# FUTURE_ML_01

##  Project Title
Sales & Demand Forecasting

##  Description
This project focuses on predicting future sales using machine learning techniques. 
The goal is to analyze past sales data and forecast upcoming demand to help in better decision-making.

##  Dataset
The dataset contains historical sales data including features like:
- Date
- Product details
- Sales quantity
- Other relevant factors

##  Steps Performed
- Data Cleaning and Preprocessing
- Handling Missing Values
- Feature Engineering
- Model Training
- Model Evaluation

##  Machine Learning Models Used
- Linear Regression
- Random Forest

##  Results
Both models were trained and evaluated to predict future sales.  
Random Forest generally provides better accuracy compared to Linear Regression.

##  How to Run the Project
1. Open the `.ipynb` file in Google Colab or Jupyter Notebook
2. Run all cells step by step
3. View the predictions and output graphs
